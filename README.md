# Market Dashboard

A Django-based stock analysis web application built on top of a custom Python analytics engine.

## Features
- Input up to 8 stock tickers
- Real-time financial data via yfinance
- Performance metrics (CAGR, volatility, drawdown, momentum)
- Clean dashboard UI
- Modular backend using a custom Stock class

## Tech Stack
- Python
- Django
- Pandas
- yfinance
- HTML/CSS

## Architecture
This web application is powered by a separate stock analytics engine that handles:
- Data ingestion
- Metric computation
- Performance evaluation
